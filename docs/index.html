@sym20/core

This package provides fundamental APIs to initialize Extensions and accessing internal data Stores in Symphony 2.0

This package provides APIs to access Symphony Chat UI Services and Chat Overlay views.

@sym20/core-conversations

This package provides APIs to access Symphony Conversation Stores

@sym20/core-ui

This package provides APIs to access generic Symphony UI Services and Views

@sym20/extension-webpack-plugin

This extension is used to pack a Symphony 2.0 extension into a .tgz file that can be deployed into a Symphony 2.0 Client instance.
